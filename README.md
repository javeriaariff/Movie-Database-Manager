# 🎬 Movie Database Manager

A C++ console-based application that manages and analyzes a movie database using custom-built data structures and graph algorithms. The system enables efficient movie searching, recommendations, shortest-path analysis, and statistical insights without relying on STL containers for the core implementation.

## Features

- Search movies by title
- Search movies by actor
- Search movies by genre
- Search movies by release year
- Search movies by rating range
- Movie recommendation system
- Shortest path between movies
- Actor-to-actor connection analysis
- Director-to-director connection analysis
- Database statistics and top-rated movies

## Data Structures Used

- Linked Lists
- Hash Tables
- Graphs
- Queues
- Dynamic Arrays

## Algorithms

- Breadth-First Search (BFS)
- Hashing
- Graph Traversal
- Sorting
- CSV Parsing

## Technologies

- C++
- Object-Oriented Programming
- File Handling

## Dataset

The project uses a CSV dataset (`movie_metadata.csv`) containing movie information such as titles, genres, actors, directors, ratings, release years, and durations.

## How to Run

1. Clone the repository.
2. Open the project in Visual Studio or any C++ IDE.
3. Place `movie_metadata.csv` in the project directory.
4. Build and run the application.
